# 💊 NaijaMeds

NaijaMeds is a Flutter-based mobile application designed to improve access to safe medication information and healthcare services. The app allows users to search for drugs, scan medications, view nearby pharmacies on a map, and manage their profiles securely.

---

##  Features

-  Home screen with overview and quick access
-  Drug search functionality
-  Drug scanning (QR/Barcode scanner)
- ️ Map integration for nearby pharmacies/locations
-  User authentication (Firebase Auth)
-  Protected routes for logged-in users
-  Reports and medical information access
-  Profile management and settings Smooth bottom navigation dashboard

---

##  Tech Stack

- Flutter (Frontend UI)
- Dart
- Firebase Authentication
- Google Maps API(need to pay first)
- Material Design UI

---

##  Project Structure
- Nice layout structure with clean code
-  Clean code
  -  Easy to understand

 ## Authentication Flow

- Users can browse basic features without login
- Protected features (Profile, Search, chat area etc.) require authentication
- If a user is not logged in, they are redirected to the **Auth Screen**
- After login/signup, they are redirected back to the selected screen
